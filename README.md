# Advent of Code 2021

Answers for the [Advent of Code 2021 challenge](https://adventofcode.com/2021) using Elixir

## Dependencies

* Elixir 1.12.3 or above

## How to run it

Once you clone this repo, run the following commands:

```shell
# Fetch all dependencies
mix deps.get

# Run the test suite
mix test --trace
```
